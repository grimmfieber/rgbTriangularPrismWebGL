### Colors can be switch by using k,y,m keys or reset with S.

>S for resetting colors.
>K for Red
>Y for Green
>M for Blue 

## why not 'R' 'G' 'B' keys instead 'K' 'M' 'Y'
### In Turkish
##### Red = Kırmızı
##### Green = Yeşil
##### Blue = Mavi 
##### Reset = Sıfırlama
